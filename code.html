The Code for Zero to 100

 
                        
                             function getValues(){
                                let startValue = document.getElementById('startValue').value
                                let endValue = document.getElementById('endValue').value

                                //Attempt to parse into Integers
                                startValue = parseInt(startValue)
                                endValue = parseInt(endValue)


                                 if(Number.isInteger(startValue) && Number.isInteger(endValue)){ 
                                    //Call generateNumbers function
                                    let numbers = generateNumbers(startValue, endValue)

                                    //Call displayNumber function
                                    displayNumbers(numbers)
                                }else{
                                    alert("Integers only")
                                }
                            }

                             function generateNumbers(startVal, endVal){
                                let numbers = [];
                                for(let i = startVal; i <= endVal; i++){
                                    numbers.push(i)
                                }
                                return numbers;
                            }

                             function displayNumbers(arrayOfNums){
                                const tableEl = document.getElementById('results')
                                tableEl.innerHTML = '';
                                for(let i = 0; i < arrayOfNums.length; i++){
                                    if(arrayOfNums[i] % 2 === 0){
                                        tableEl.innerHTML+=
                                        `${arrayOfNums[i]}`
                                    }else{
                                        tableEl.innerHTML += 
                                        `${arrayOfNums[i]}`
                                    }
                                }
                            }
getValues()

The getValues() function is responsible for retrieving the numbers the user provides in our input fields and parsing them as integers. If both inputs provided by the user are both numbers than we will call the generateNumbers() and displayNumbers() functions described below.


generateNumbers()

The generateNumbers() function takes two parameters which are the starting value and the end value. The function uses a for loop to iterate through the numbers between the values the user provided and pushes those numbers to an array. The array is then returned.


displayNumbers()

The displayNumbers function takes one parameter which is an array of integers. The function retrieves the table we made in HTML using the id and then sets the innerHTML to be empty so if the user spams the button it does not constantly add to the old table. Then using a for loop to iterate through the array of numbers (these are all the numbers between the 2 values the user provided) I seperate the numbers into two categories using if statements. If the number is divisible by two then I want the number to appear bolded otherwise the number wilk appear normally



Conclusion

The javascript code written for this app takes the user input and finds all numbers between those two. I then determine if when the numbers between what the user entered are divisible by 2 and if they are I apply a strong tag, otherwise no style is applied. The array of numbers is then added to table.
